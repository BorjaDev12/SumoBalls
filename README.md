Pelea de Titanes
Descripción
Este juego consiste en que dos jugadores están peleando por su vida encima de una platafora. Tienen dos opciones, la primera de ellas es que uno de ellos decida suicidarse y tirarse de manera voluntaria de la plataforma y uno de ellos sobrevive. La otra opción es que los dos decidan confrontarse dando lugar a que ambos choquen y se produzca una explosión.
Añadidos
Se añadieron partículas que simulen una explosión
Se añadió una skin de roca y otra de bomba nuclear a cada uno de los jugadores
Se añadieron los correspondientes scripts para que todo funcione de manera correcta
